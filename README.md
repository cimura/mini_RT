# mini_RT

引数は以下のような感じ
```
A 0.2 255,255,255
C 0,0,0 0,0,1 70
L 0,0,-3 0.6
sp 0,0,10 7 0,0,255 WOOD(材質名) N:"textmp.png"(テクスチャマップ)
```

材質名一覧
・GLASS
・IRON
・SILVER
・WOOD
・WATER

テクスチャマップの指定方法
(マップの種類):"(マップのパス)"
テクスチャは指定しなくても良い。

マップの種類
・N 法線マップ （でこぼこ）
・C カラーマップ （色）
